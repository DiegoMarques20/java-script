Somando duas variáveis

  • Para somarmos duas variáveis utilizando o prompt, precisamos declarar como number os itens digitados, por padrão o prompt entende que tudo digitado é uma string.

  • Para isso usamos o Number.parseInt(n) (Neste caso retorna número inteiro) ou Number.parseFloat(n) (Neste caso retorna número quebrado, 1.4, 4.5)

  • Então para esse nosso exemplo ficaria assim:
    var num = Number.parseInt(window.prompt('Digite um número'))
    var num2 = Number.parseInt(window.prompt ('Digite outro número'))
    var s = num + num2
    window.alert('A soma dos números digitados é ' + s)

  • Para passarmos um valor número para string, ao contrário que fizemos, utilizamos String(n).
