Guardando dados

  • Quando usamos o window.prompt como vimos anteriormente, abre uma janela no navegador para o úsuario digitar algo.

  • Podemos armazenar esse dado dentro de uma variável, por exemplo: var nome = window.prompt('Qual o seu nome?')

  • Depois desse armanezamento, podemos usar um window.alert('É um grande prazer te conhecer, ' + nome + '!'). Neste caso contatenamos
