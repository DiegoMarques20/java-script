Funções

  • Conceito de Funções: São acões executadas assim que são chamadas ou em decorrência de algum evento. Uma função pode receber parâmetros e retornar um resultado.

  • No exemplo abaixo, utilizamos uma função para descobrir se o número dentro de n é par ou impar. Neste caso chamamos a função e dentro dela inserimos um if e else e depois com console.log imprimos o resultado na tela.

  • let res = parouimpar(78)
    function parouimpar (n) {
    if (n%2==0){
    return"O número é Par"
    } else {
    return"O número é impar"
    }
    }
    console.log(res)
