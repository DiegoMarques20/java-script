Null e Undefined

  • Undefined é quando não passamos um valor para uma variável, então ela é indefinida e não conseguimos seguir em frente e nem coletar dados.

  • Null é quando não sabemos o dado de uma variável naquele momento e propositalmente identificamos ela como Null.
  • Variável Const:

    Uma das vantagens do uso de const é o conceito de imutabilidade, que quer dizer "manter o mesmo valor" ou "não mudar".No JavaScript veremos muito o uso de const por causa desse conceito de imutabilidade.
