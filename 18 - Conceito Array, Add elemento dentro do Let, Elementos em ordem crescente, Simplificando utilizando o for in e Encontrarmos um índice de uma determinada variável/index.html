Variáveis Compostas

Array

São variáveis compostas que possuem vários elementos e esses elementos são identificados como índices, exemplo [0,1,2,3..]

Para identificarmos essa variáveis usamos o LET inves do VAR, então ficaria assim let aut = [Jeep,Fiat,Audi,Ferrari]

Add elemento dentro do let

  • Para add um elementro detro do let, fazemos como no exemplo abaixo, colocamos entre colchetes o índice a ser add o sinal de igual e o valor. no Exemplo abaixo estamos add no índice 1 o número 6.


  • let num = [4]
    num[1] = 6
    console.log(`Nosso vetor é ${num}`)

    • Add automaticamente o elementro dentro da váriavel.

  • Quando não soubermos qual índice está vázio para passar essa informação igualmente acima, utilizamos o num.push(6). Neste caso é add automaticamente o elementro dentro da váriavel.


  • let num = [4]
    num.push(6)
    console.log(`Nosso vetor é ${num}`)

    • Elementos em ordem crescente

  • Quando quizermos colocar os elementos em ordem crescente, usamos o num.sort() E para mostrarmos um elemento em específico usamos o console.log(num[2]) Onde num é o nome da variável e o 2 é o índice que precisamos

    • Simplificando utilizando o for in

  • Podemos simplificar utilizando o for in, conforme exemplo abaixo:

    let num = [4,,8,7,5]
    num.push(6)
    num.sort()
    for ( let pos in num) {
    console.log(`A posição ${pos} tem o valor ${num[pos]}`)
    }

    • Encontrarmos um índice de uma determinada variável

  • Para encontrarmos um índice de uma determinada variável, utilizamos o num.indexOf(audi)
    Com isso ele retorna o número do índice do elemento.
    let auto = ["ferrari","audi","BMW","Mercedes"]
    let pos = auto.indexOf("audi")
    console.log(`O carro audi está na posição ${pos}`)
