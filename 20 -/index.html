Objetos em JavaScript

  • Utilizamos Array para coisas simples e intermediárias, para processos mais avançados, devemos utilizar objetos de acordo com a necessidade. Para utilizarmos objevos, invés de passarmos colchetes que fazemos com array, vamos utilizar chaves. {}

  • Objetos conseguem salvar variáveis e também criar funções dentro dela, no exemplo abaixo criamos uma função de diminuição e apresentação de peso de um usuário.


  • let amigo = {nome:'Diego', sexo:'M', peso:85.0, emagrecer(p){

    this.peso += p
    }}
    amigo.emagrecer(-2)
