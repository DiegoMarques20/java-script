Eventos DOM

  • Para usarmos o Dom e fazer interações com o HTML, existem algumas ações exemplo quando o usuário utiliza o mouse. Segue alguns exemplo abaixo:

  • onmouseenter = Quando o usuário entre com o mouse dentro do campos.
    onmouseout = Quando o usuário sai com o mouse do campos.
    onmousemove = Quando o usuário move o mouse dentro do campo.
    onmousedown = Quando o usuário clicar e segura dentro do campos .
    onmouseup = Quando o usuário solta o click dento do campo.
    onclick = Quando o usuário clicar dentro do campo.
  • Para utilizarmos o mecanismo acima, primeiro informamos dentro da tag os componentes acima conforme nesse exemplo:
    div id="area" onclick="clicar()" onmouseenter="entrar()"

  • Depois abrimos um Script para utilizar os comandos, como passamos o id com o nome de area, usamos o getElementById, como no exemplo abaixo:
    var a = window.document.getElementById('area')

    a.innerText = 'Clicou!'
    a.style.background = 'red'}
