Usando mais condições com Else IF

  • Podemos inserir várias condições dentro de um código, para isso utilizamos o else if, conforme exemplo abaixo:
    var idade = 22
    if (idade < 16) {
    console.log ('Não Vota')
    } else if (idade < 18 || idade > 65 ) {
    console.log ('Voto Opcional')
    } else {
    console.log ('Voto Obrigatório')
    }
