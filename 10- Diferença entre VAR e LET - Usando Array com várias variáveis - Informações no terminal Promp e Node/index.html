Imprimindo Informações na Tela (Terminal Prompt ou Node)

Diferença entre declarar variáveis do tipo VAR E LET

  • Para declararmos uma variável e imprimir essa variável no prompt ou node, utilizamos o seguinte código:


    var programador = "Diego";
    console.log(programador);

  • Diferença entre declarar variáveis do tipo VAR e LET

  • Variaáveis do tipo VAR são mutáveis, ou seja conseguimos alterar mudando o resultado da variável.

  • Variável do tipo LET não pode ser alterada o seu valor, depois de declarado.
  • Utilizando Array: Utilizamos Array quando temos várias variáveis, neste caso informamos conforme esse exemplo:
    var estados = ["Minas Gerais" ,"Rio de Janeiro", "São Paulo", "Bahia"];
  • Para solicitarmos uma variável específica, usamos o
    console.log(estados[0]); (Dentro do colchetes chamamos o número da variável, lembrando que começa sempre do 0, neste caso o código chamaria Minas Gerais

  • Conseguimos alterar os dados dentro de um Array, conforme no exemplo abaixo: (Neste caso C# é subistituido por Python)
    var linguagens = ["JavaScript", "PHP", "JAVA", "C#"];
    var linguagem = "Python";
    linguagens[3] = linguagem;
