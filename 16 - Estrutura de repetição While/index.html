Repetições

  • Para estrutura de repetições, utilizamos o While (Enquanto). Segue um exemplo simples abaixo, se uma estrutura while onde vamos escrever o tudo bem por 6 vezes, lembrando que o c++ add 1 au número.

  • Podemos ter duas estruturas While, uma conforme o exemplo onde ele valida uma condição e executa e a outra é onde ele executa o código e depois valida a condiçao, para esse última, utilizamos o do. Do {} While (Condição)

  • var c = 1
    while ( c <= 6) (Enquanto 6 for menor ou igual a c, execute o comando abaixo) {
    console.log('Tudo bem?')
    c++
    }
    Resultado =
    Tudo bem?
    Tudo bem?
    Tudo bem?
    Tudo bem?
    Tudo bem?
    Tudo bem?

  • var c = 1
    do {
    console.log('Tudo bem?')
    c++
    } while (c <=6)
