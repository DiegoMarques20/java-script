Aprendendo a usar o DOM

  • O DOM serve para selecionar e alterar elementos dentro de uma página, utilizando JavaScript. Usamos o Get Element podemos escolher se vamos selecionar por Marca, ID, Nome, Classe ou Seletor.

  • Por seletor é o mais atual e talvez mais recomendado.

  • Exemplo do uso do Seletor, mudando a cor do texto e o fundo.
    var d = window.document.querySelector('div.oi')
    d.style.color = 'blue'
    d.style.background = 'white'

  • Exemplo usando GetElement para alterar um texto de uma tag p, como temos mais de uma tag p, informamos o número da tag conforme no exemplo abaixo.
    var e = window.document.getElementsByTagName('p')[0]
    e.innerText= 'Olá'
